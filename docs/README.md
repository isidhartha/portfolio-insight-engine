# Documentation assets

This folder contains static example assets for the current portfolio email design.

| File | Purpose |
| --- | --- |
| `example.html` | Standalone preview of the email using a fixed fictional sample portfolio and a sample AI-style weekly summary |
| `example.png` | Rendered image preview of `example.html` |
| `chart-example.png` | Weekly holding contribution bar chart example |
| `diversification-chart-example.png` | Portfolio diversification doughnut chart example |

The examples are generated from a fixed fictional sample snapshot so the documentation stays stable even when live market data changes and does not expose personal holdings.
